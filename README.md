# Java-ATM-System

Javaで実装された、本格的な銀行口座管理システムです。正確な数値計算と堅牢なエラーハンドリング、そして自動テストによる品質保証を特徴としています。

主な機能
- **認証システム**: 暗証番号によるログイン機能と、3回連続ミスによるロック機能を搭載。
- **正確な金額計算**: `BigDecimal` を採用し、金銭計算における誤差を排除。
- **取引履歴管理**: 預入れ・引き出しの履歴を日時とともに自動記録（`LocalDateTime` 使用）。
- **堅牢なエラーハンドリング**: 独自例外 `InsufficientBalanceException` を実装。

使用技術
- **言語**: Java (SDK 25)
- **テスト**: JUnit 5 (単体テスト実装済み)
- **管理**: GitHub Projects (タスク管理)

テスト実績
